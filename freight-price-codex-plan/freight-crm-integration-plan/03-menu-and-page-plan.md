# 03 菜单与页面规划

## 1. 菜单边界

建议遵循“履约归货代、财务归 CRM”的菜单边界。

### 国际货代

- 业务单管理

### CRM 系统

- 客户管理
- 合同管理
- 回款计划
- 回款管理

## 2. 业务单页面改造

### 列表页

新增展示字段：

- 合同编号
- 合同名称

新增筛选字段：

- 合同编号
- 合同名称

### 新建 / 编辑表单

新增联动逻辑：

1. 先选择客户
2. 再按客户过滤合同下拉
3. 选择合同后自动带出合同编号与合同名称快照

### 详情页

在现有详情页基础上增加“合同摘要”区域：

- 合同编号
- 合同名称
- 合同金额
- 已回款
- 待回款
- 查看合同详情

## 3. CRM 合同详情页改造

建议在合同详情页增加一个 Tab：

- `关联业务单`

该 Tab 展示字段建议：

- 业务单号
- 业务类型
- 运输方式
- POL
- POD
- 当前状态
- ETD
- ETA
- 创建时间

## 4. CRM 客户详情页改造

客户详情页新增两个 Tab：

- `银行账户`
- `营业执照/资质`

### 银行账户 Tab

展示内容：

- 户名
- 开户行
- 银行账号
- Swift
- 币种
- 默认账户
- 备注

### 营业执照 / 资质 Tab

展示内容：

- 资质类型
- 证照编号
- 公司名称
- 到期日期
- 附件
- 备注

## 5. 合同与回款页面联动建议

### 合同表单

建议后续支持：

- 选择客户默认银行账户
- 查看客户资质摘要

### 回款表单

建议后续支持：

- 选择实际收款账户
- 展示收款账户币种
